# Num Buddy
 Resolva problemas matemáticos de maneira simples.
 - Funções Afim
 - Funções Quadráticas
 - Logaritmos
 - Fibonacci
 - Progressão Aritmética
 - Progressão Geométrica
 - Medidas de Dispersão
   
# Pré-requisitos
Python versão 3 ou superior

# Instalação
```
git clone https://github.com/hitalloazevedo/num-buddy
cd num-buddy
```

# Como executar?
```
python3 main.py
```
ou
```
python main.py
```
